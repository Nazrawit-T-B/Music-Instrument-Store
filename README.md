# Music-Instrument-Store
A clean, simple, and user-friendly online music store built using modern web technologies. <br>
This project focuses on a smooth browsing experience, responsive design, and intuitive UI for exploring, previewing, and purchasing music online.<br>

## Features

🎻 Musical Instrument Browsing

Explore instruments in stock at the store

Clean card-based layout

Search bar with instant filtering

🛒 Shopping & Checkout

Add instruments to cart

View cart summary with dynamic price updates

Responsive checkout page (UI only, no backend)

📱 Fully Responsive

Mobile-first design

Adapts smoothly to desktop, tablet, and mobile screens

💡 UI/UX Highlights

Minimalistic and modern interface

Smooth animation transitions

Accessible color contrast & readable typography

🛠️ Tech Stack
Frontend

HTML5

CSS3 (Flexbox + Grid)

JavaScript (ES6+)<br>
